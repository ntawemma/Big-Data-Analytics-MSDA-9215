## Hands-on Neo4j: US Road Network Analysis

This repository contains the implementation and analysis of a large-scale US road network using Neo4j and Python.

### Contents
- Neo4j data import and Cypher queries
- Graph analytics (degree, shortest path, centrality)
- Python-based Plotly dashboard
- Final academic report (Overleaf)

### Tools Used
- Neo4j Desktop (5.x)
- Cypher
- Python (NetworkX, Plotly, Neo4j Driver)
